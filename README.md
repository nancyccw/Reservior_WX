# Reservior_WX
一个水库监测项目的微信公众号项目，基于微信公众号平台API，使用thinkPHP5.0框架开发。其他使用到的技术包括：js，frozenui，echart等

在绘制大坝浸润线时，使用echarts的频率很高，涉及到：
1. x轴为value类型的用法
2. 双Y轴的用法
3. 自定义图形的绘制，包括line、text、polygon等的组合
4. 贝塞尔曲线（未完成）
